General Notes for Backend Team
📋 Standards & Conventions
  • Authentication: All endpoints except /auth/login, /auth/register, and /auth/forgot-password require Authorization: Bearer <token>
  • Error Format: All errors must return { error: { code: int, message: string } }
  • Timestamps: All date/time fields must use ISO 8601 format
  • Pagination: Use page (starts at 1) and limit (default 20). Always return total in paginated responses
  • Images: All image uploads via multipart/form-data
  • Sensor Status: Backend is responsible for calculating normal / warning / critical based on threshold ranges defined above
  • suggestedTreatment: The scan detect endpoint must return a linked treatment plan when disease is detected — this powers the "Start Treatment" and "Remind Me Later" flows in the app
